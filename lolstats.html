Project propsal of A site that uses javascript, html, css to create a functional webpage for users to search summoners from the popular game league of legends and display the stats of their games. Create a library that uses node.js to export lolwrapper function containing Riot Games api to use for the webpage. Home page- Select region Na, Kr, Eu, Br, ect.. Search Summoners name -> returns 10 most recent games of that summoner. 2nd page- Statistics for highest winrate of champions and most popular builds for champions.
